Tiny BBS for sample

Requirement
-----------

Docker and Docker Compose

How to run
----------

Edit sample.env and rename it to .env. Then, execute the following commands.

```console
./docker-compose-wrapper.sh up -d
```

View at: <http://localhost:4567/>
